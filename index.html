Portfolio

Recent Works
ReciPy
Mobile meal-generating app built using Python, Typescript, React Native, and AWS. The app offers tailored meal suggestions complete with comprehensive recipes, which are customized based on the user's preferences, dietary restrictions, and caloric boundaries. It includes barcode scanning to manage a user's pantry, expiration tracking, and push notifications to reduce food waste. The app also employs a recommender system based on K-means clustering to suggest meal options to users, and a social component managed through AWS to facilitate interaction and content sharing among users.
Employee Task Management System
Full-stack employee and task management web application built using Node, Express, React, Redux, PostgreSQL, and Sequelize. With all CRUD operations integrated, it makes managing data a breeze. The app hosts a comprehensive array of features, including real-time error messages, many-to-many relationships between models, handler functions for routing, Redux for state management, and client-side routing with React-Router. The application is a powerful solution for anyone looking to streamline employee and task management.
DCA Investment Projector
iOS application built using Swift and UIkit that acts as a powerful dollar cost averaging calculator for investors. The app leverages an API service that lets users search for stocks and offers them with projections of potential earnings based on their investment strategy, including initial investment, monthly adjusted investment, and the initial date of investment. With a user-friendly interface, the application is easy to navigate and perfect for anyone looking to invest in stocks over a longer period.
